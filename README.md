# 瑞思壁纸-Ris_WallPaper
## Windows桌面壁纸软件
由PYLXU(腾瑞思智)开发，在Github使用GPLv3协议开源
[请点击这里下载最新版本的构建](https://github.com/PYLXU/Ris_WallPaper/releases)

## 基本说明
本程序支持使用WMP实现桌面动态壁纸，与设置系统壁纸的功能，支持动态调整是否启动引擎
